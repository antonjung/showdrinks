# overview

a system to manage the pre purchasing of drinks for before, during and after a show.

## a web page which allows you to 
* create records for items for sale and prices.
* set up the valid dates for the sales to be for - typically show dates
* set up the name of the show
* set up sessions - eg , before, interval and after the show
* set up the places where drinks can be found once prepared.
* set up the details for sumup account so the pwa can make payment

* review the drinks ordered for the valid days and sessions as a tick list
* mark drinks as prepared and where they can be found as a drop down

## a pwa 
* scan a QR code to load the pwa
* enter users name
* order drinks for a date and "session" (before during after)
* make payment for the drinks
* see if the drinks are ready and where they will be
